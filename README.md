# Online-Retail-Application
JavaFX application incorporate with CSS and SQLite

1. This program utilize sqLite as data storage, located in \gamestore.db
2. To launch the program, go to Coursework\src\main\java\com\example\project and launch the file "GameZone"
3. This program comes with a built in admin account:
    username: admin
    password: admin
4. a database with initial data will be launched, if an empty database is required, simply delete the gamestore.db
   and a new database will be created upon launching
